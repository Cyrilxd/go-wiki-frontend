# Go-Wiki Frontend Plugin

**WIP: Work in Progress**

This is the frontend plugin for Go-Wiki.

## Development

### Environment

___Web-App___
* **NodeJS** 8.x LTS

___Go-Plugin___
* **Go** 1.10.x
* **Dep** 0.5.x
* **go-bindata**

### Build application

You can use the included Makefile. Enter `make clean web_app go_plugin` to build everything.
